# TaskFlow Platform

A full-stack project management platform built with Spring Boot, React, and MySQL.

## Features

- User authentication and authorization
- Team management
- Project management
- Task assignment and tracking
- Comments and activity logs

## Tech Stack

### Backend
- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL

### Frontend
- React
- Axios
